# Magoya - Frontend Dev Test

## Setup environment

* **node.js v12.16.2**
* **npm**

Note: Running the application in **node.js v8.16** was found to cause issues

## To install

```bash
npm install
```

## To run dev server

```bash
npm run dev
```
## To build

```bash
npm run build
```

## To start the production server locally

```bash
npm run start
```
